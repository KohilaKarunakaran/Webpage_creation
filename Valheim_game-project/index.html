VALHEIM­RAFT
The mod for Valheim

Have you ever wanted to create your own ships in
Valheim?

Valheimraft mod allows you to create your own ship! As you may see by the name it is very much inspired by the game Raft!

With this players can build a basic raft and use it as a building platform, adding as many construction objects as they prefer to get the ship looking exactly as they want
